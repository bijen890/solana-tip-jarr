# solana-tip-jar
Our first solana D-app. A simple tip jar

Solana Tip Jar
Our first decentralized application on Solana! 

What it does
A simple tip jar where anyone can connect their Phantom wallet and send SOL tips to support creators.

Why We built it
As a beginner learning Solana development, We wanted to understand how wallet connections and transactions work without needing to write complex smart contracts.

How to use
Connect your Phantom wallet (Devnet)
Click a preset amount (0.01, 0.05, 0.1 SOL) or enter custom
Approve the transaction
SOL is transferred instantly!

Tech Stack
Solana Web3.js
Phantom Wallet
HTML/CSS/JavaScript
Deployed on Vercel
